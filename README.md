# News App with Navigation
 
Simple Android application for test of my android backend skills.

# Description

This application is my playground to clutch MVVM arthitecture, Navigation components and various state of art technologies. 

# Design

The design adopted follows the MVVM with clean architecture design patterns. 

# Libraries

1. ViewBinding
2. Coroutines
3. Hilt
4. ViewModel
5. Flow
6. Navigation
7. Fragments
8. OkHttp
9. Retrofit2
10. Glide
11. Moshi
